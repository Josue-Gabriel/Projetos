Aplicação para armazenamento de arquivos em um banco de dados, contendo informações como tipo do arquivo, nome e descrição. Desenvolvido em PHP para o

gerenciamento de informações que estão trafegando no uso do sistema.



Possuindo campo de busca de arquivos e suas informações salvas no banco de dados para análises gerais, podendo também fazer a remoção das informações nos registros.



Todos os campos sitados a cima podem ser acessados no menu inicial após efetuar o login. Tendo a opção também de encerrar a sessão.



Possuindo sistema de login com criptográfia de senha em três camadas, onde em cada camada o senha passa por um tipo de criptográfia diferente, gerando valores

aleatorios e descriptografando de uma só vez pelo o próprio sistema.
